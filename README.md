ภารกิจที่ 2: สร้างและรัน Spatial Model

รัน Multi-Criteria Analysis ใน GEE โดยต้องมีขั้นตอนเหล่านี้อย่างชัดเจน:

2.1 การ Normalize แต่ละ Factor (อธิบายวิธีและเหตุผล)

ใช้ Min-Max Normalization

สูตร:

x_norm=(x-x_min)/(x_max-x_min )

เนื่องจากแต่ละ factor มีหน่วยต่างกัน (HAND = เมตร, Slope = องศา, NDVI = -1 ถึง 1, TWI = ไม่มีหน่วย) ถ้าไม่ Normalize factor ที่มีค่าสูงกว่าจะครอบงำผลลัพธ์โดยอัตโนมัติ

และมีการ Invert (กลับค่า) บางปัจจัยเนื่องจากบางปัจจัย "ค่าสูง = เสี่ยงต่ำ" ต้องกลับเป็น 1 - normalized

Factor	ทิศทาง	เหตุผล

HAND	Invert	ต่ำ = ใกล้แม่น้ำ = เสี่ยงสูง

Slope	Invert	ราบ = น้ำระบายช้า = เสี่ยงสูง

NDVI	Invert	ไม่มีพืช = ดูดซับน้ำน้อย = เสี่ยงสูง

TWI	ปกติ	สูง = ดินชื้น = เสี่ยงสูง

2.2 การกำหนด Weight และ Combine (สมการต้องชัดเจน)

สมการ WLC:

Flood_Risk=0.35×HAND_risk+0.25×Slope_risk+0.20×NDVI_risk+0.20×TWI_risk 

- HAND (Height Above Nearest Drainage) 0.35 

- Slope (ความลาดชัน) 0.25 

- NDVI (ความหนาแน่นพืชพรรณ) 0.20 

- TWI (Topographic Wetness Index) 0.20
